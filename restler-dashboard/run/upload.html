Step 1 of 3: Upload

  1. Run
RESTler requires a valid YAML file so that the engine can parse and eventually run the fuzzing process on it.

To upload a valid YAML spec choose ONE of the following options.
Option 1: Local YAML Upload

Choose and upload a YAML file from your computer.


Option 2: Cluster VIP Upload

Input a valid Cohesity Cluster VIP in the following text box then click upload.

This will automatically submit a unified Cohesity YAML corresponding to the specified cluster.
